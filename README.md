# NCKU-parser

Refined the data of NCKU

##Usage

- Course Parser:
Course Parser is a Parser to parse courses of NCKU

```
python course-parser/courseParser.py
```

- Club Parser
Club Parser is a Parser to parse club info of NCKU

```
ruby club-parser/club-parser.rb
```
